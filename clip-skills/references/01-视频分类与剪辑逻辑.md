# 01 · 视频分类与剪辑逻辑

> 目的：让 Agent **先判断视频属于哪一类**，再套用对应的剪辑骨架。类型判断错，后面全错。

## 一、如何判断类型（输入信号 → 类型）

从用户简述 + 素材构成判断，不确定就反问：

| 信号 | 判定类型 |
|:---|:---|
| 有一条贯穿始终的人声讲解/口播，画面多为讲解者 + 演示 | **口播教程** |
| 大量短镜头 + 一首主音乐，强调画面与节拍 | **混剪 / 卡点** |
| 第一人称记录、生活/旅行片段、有旁白或现场声 | **Vlog / 样片日记** |
| 同一时间多机位拍摄、对话/问答 | **采访 / 多机位** |
| 竖屏、强包装、文字动效多、综艺感 | **综艺竖屏包装**（第10集） |
| 文字/图形要贴合并跟随画面物体（带透视） | **运镜 / 跟踪贴合**（第11集） |
| 成片 > 10 分钟、有明确章节 | **长视频**（叠加在上面某一类之上） |

## 二、各类型剪辑骨架（Agent 决策版）

### 口播教程（第1/2集）
- **骨架**：声音先行。口播精剪成 A-roll 时间轴 → 按字幕语义在上层贴 B-roll。
- **决策**：信息密度决定镜头时长；气口按规则处理（见 `02`）；硬切用放大/万能 B-roll 遮蔽（见 `03`）。
- 详见 `02-音频驱动剪辑流程.md` + `03-素材AB分类与替换.md`。

### 混剪 / 卡点（第3集）
- **骨架**：先铺主音乐 → 检测节拍/重音时间戳 → 在节拍点切换镜头。
- **决策**：素材按景别分（远/中/特）；用**景别递进**（远→中→特）制造层次；相似构图/动作/颜色做**匹配剪辑**衔接；适当**切分节拍**打破均匀、制造变化。特效是调味料不是主菜。

### Vlog / 样片日记（第14集）
- **骨架**：乱序思维导图（先不排序、把所有素材/想法扔上去 → 用箭头标前后呼应 → 整理成正式思维导图确立主题）→ 提炼主线（可借大模型：剪映提取文字 → 喂 LLM 总结编排，作参考而非最终架构；用"智能镜头分割"拉片分析参考片节奏）→ 分段式剪辑，每段独立剪好再串。
- **决策**：片段串联用"旁白衔接"或"节奏点衔接（B-roll+音乐小混剪）"；多主题切换点对齐时钟理论 0/3/6/9 点；调色常用 S 曲线 + 色彩克隆统一调性（见 `06`）。

### 采访 / 多机位（第12集）
- **骨架**：多机位按音频波形对齐合板 → 导播式粗剪（在说话人开口处切到对应机位）→ 加人名条 → 提取金句。
- **决策**：切换跟着"谁在说、说的内容重不重要"走；长素材开代理保证流畅；切错机位就重选机位，不必重切。

### 综艺竖屏包装（第10集）
- **骨架**：竖屏布局 → 智能镜头分割定切点 → 文字/数字动效 + 二维跟踪贴合画面 → "踩梯子"递进。
- **决策**：信息密度定镜头时长（开头约 1 秒保留存、信息多放长到 4 秒）；化繁为简只突出核心信息，必要时不加转场不加音乐集中注意力；文字二维跟踪选"占比最大的那一帧"对齐高对比特征点、取消"缩放/距离"保持字号统一。详见 `10-跟踪与动效包装.md`。

### 运镜 / 跟踪贴合（第11集）
- **骨架**：预览标记要贴合的点位 → 文本先打包复合片段 → 平面跟踪四点对齐物体 → 在占比最大帧重对 → 平面消失处切分加渐隐。
- **决策**：要文字"长"在物体表面、随透视同步 → **平面跟踪**（三维）；只是文字随物体位移 → **二维跟踪**。跟踪耗算力，一律**先粗剪后做、做完导出备份**（别用复合片段打包多跟踪轨，易卡死）。详见 `10-跟踪与动效包装.md`。

### 长视频（第13集，叠加层）
- **骨架**：开头钩子三步（抛问题→揭主题→混剪预告）→ 波峰制起伏 → 分段式剪辑 → 时钟理论布爆点 → 快速结尾。
- **决策**：时长选 13–23 分钟（高节奏）或 50+（深度），避开 35–40 分钟（完播差）；后段节奏要更快。详见 `05`。

## 三、通用决策原则

1. **信息密度 = 镜头时长的标尺**：信息少则快切，信息多则放长，让观众跟得上。
2. **先骨架后装饰**：先把叙事/声音骨架立住，再加 B-roll、转场、特效、调色。
3. **耗算力操作后置**：抠像（`09`）、跟踪（`10`）、补帧/变速（`08`）等放到粗剪定稿后做，做完先导出备份（性能与代理、导出参数见 `12`）。
4. **分段式优先**：长内容拆成"多个有趣短段"分别剪好再总装，避免一口气从头剪到尾导致前后脱节。
