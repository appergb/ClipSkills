# 其他 NLE 适配技能

课程基于剪映演示，但本套技能要能迁移到任意 NLE。本技能不堆操作细节，而是给**迁移地图**：核心动作在各软件叫什么、能否被 Agent 驱动、查什么。

## 一、自动化能力分级（决定 Agent 能做到哪一步）

| 软件 | Agent 可自动化程度 | 接口 |
|:---|:---|:---|
| **Palmier Pro** | 高（直接操作时间线） | 原生 MCP（见 `editor-palmier-pro`） |
| **DaVinci Resolve** | 中高（脚本驱动） | 官方 **Python/Lua 脚本 API**（DaVinciResolveScript），Fusion 节点可脚本化 |
| **Premiere Pro** | 中（脚本驱动） | **UXP / ExtendScript**，QE DOM；可批处理 |
| **剪映 / CapCut** | 低（基本靠人手） | 无公开自动化 API → Agent **产出操作清单/参数表给用户执行** |

> 结论：剪映类无 API 时，Agent 的产出形态是"**结构化操作清单 + 参数**"（切点、轨道安排、参数值），由用户在软件里照做；Premiere/达芬奇则可生成脚本自动执行。

## 二、核心动作迁移对照表

| 知识内核动作 | 剪映/CapCut | Premiere | DaVinci |
|:---|:---|:---|:---|
| 分割/裁剪 | 分割 `Ctrl/Cmd+B` | Razor / Ripple Trim | Blade / Trim |
| 多轨叠 B-roll | 副轨/画中画 | 上层视频轨 V2+ | 上层轨 / Edit 页 |
| 口播语义精剪 | 智能剪口播 | Text-Based Editing（转写后删字） | （转写后手动） |
| 自动卡点 | 自动卡点 | 标记 + 节拍（或插件 BeatEdit） | 标记节拍 |
| 关键帧/变速 | 关键帧/曲线变速 | Effect Controls 关键帧 / Time Remap | Keyframes / Retime |
| 抠像 | 智能抠像/色度抠像 | Ultra Key | Magic Mask / 3D Keyer |
| 一级调色 | 调节面板/色轮 | Lumetri Color | Color 页 一级轮 |
| 套 LUT | 调节>LUT | Lumetri > Creative LUT | Color > LUT |
| 示波器 | 内置示波器 | Lumetri Scopes | Scopes（Color 页） |
| 跟踪 | 平面/二维跟踪 | Mocha / 内置跟踪 | Fusion Planar Tracker |
| 关键帧/曲线动画 | 关键帧◆ + 曲线 `Option/Alt+K` | Effect Controls 关键帧 | Keyframes / Inspector |
| 多机位 | 新建多机位片段 | Multi-Camera | Multicam |
| 导出 | 导出 | Media Encoder | Deliver 页 |

> **决策在内核、操作在本表**：每个动作"何时做、依据什么、输出什么"的判断分别在 `jianji-knowledge` 的 `07`(关键帧)/`08`(变速)/`09`(抠像蒙版)/`10`(跟踪动效)/`11`(多机位)/`12`(导出) 各册；本表只解决"换到该软件后这个动作叫什么、能否脚本化"。

## 三、Agent 该去查找哪些信息（换软件 checklist）

接到一个新 NLE 时，按序查清：

1. **有无自动化接口**：官方脚本 API / 插件 / 命令行？（决定能否自主执行）
2. **工程文件格式**：能否读写其工程或交换格式（XML/AAF/EDL/FCPXML）？很多软件可经 NLE XML 互导（Palmier 也导出 NLE XML）。
3. **核心动作的等价命名**：用上表逐项确认"分割/叠轨/调色/导出"在该软件叫什么。
4. **转写/智能剪辑能力**：是否有 Text-Based Editing / 智能剪口播，决定 `02` 流程怎么落地。
5. **快捷键/参数默认值**：产出操作清单时给准坐标与数值。
6. **示波器与调色模型**：确认能否量化判断（对接 `06`）。

> 查找来源：该软件官方文档（经 Context7）、脚本 API 参考、社区教程。**不确定就查，不臆造接口与快捷键。**

## 四、产出形态

- **可脚本化（Premiere/达芬奇）**：生成脚本 + 说明，能自动执行的就自动执行。
- **不可脚本化（剪映）**：产出"导演级操作清单"——按时间线顺序列出每一步动作、轨道、参数，让用户照单操作；并标注黑场待裁位置。

## 五、剪映专业版速查表（产出操作清单时的数值依据）

课程基于剪映专业版（CapCut Desktop）。给剪映用户产出操作清单时，按下表给准快捷键与参数，让用户照做不犯错。

### 快捷键

| 功能 | Mac | Windows |
|:---|:---|:---|
| 分割素材 | `Command+B` | `Ctrl+B` |
| 打包复合片段 | `Option+G` | `Alt+G` |
| 关键帧变速曲线 | `Option+K` | `Alt+K` |
| 预览轴检查（抠像） | `S` | `S` |
| 禁用轨道 | `V` | `V` |
| 向右全选（平移腾空隙） | `]` | `]` |
| 放大时间线到帧级 | `Command+滚轮` | `Ctrl+滚轮` |
| 预览设入/出点 | `I` / `O` | `I` / `O` |

### 调色参数（默认值均为 0，按需增减）

色温 / 色调 / 饱和度 / 亮度 / 对比度 / 高光 / 阴影 / 白色 / 黑色；色轮四轮＝阴影/中间调/高光/偏移。

### 导出参数（对接 `12-导出与交付`）

| 参数 | 取值 |
|:---|:---|
| 分辨率 | 1080P(1920×1080) / 4K(3840×2160)，跟随素材 |
| 帧率 | 24/25/30/60，与素材一致 |
| 码率 | 选"推荐"或按平台参考表，不拉满 |
| 封装/编码 | MP4 / H.264（默认）；体积敏感用 H.265 |
| 音频 | AAC 192(语音)/256/320 kbps，采样率 48000Hz |
| 色彩空间 | Rec.709 |

> 其他 NLE（Premiere/达芬奇）的等价快捷键与参数**先查官方文档再写进清单**，不臆造。
