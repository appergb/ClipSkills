# 02 · 音频驱动剪辑流程

> 这是整套流水线的技术核心：**先有声音骨架，再贴画面**。
> 对应课程第2集（口播精剪）与第4集（音频精修）。把"看波形手动切"重写为"按字幕时间戳算切点"。

## 一、流程总览

```
提取主音轨 → 转写带时间戳字幕 → 识别断点(气口/句界) → 定切入点 → 时间轴对齐 → 输出切点/对齐清单
```

每一步的输入输出都应是**结构化数据**（时间戳列表），方便交给 editor 技能落地。

## 二、Step 1：提取音频 & 转写

- **提取**：从 A-roll（口播/旁白）分离主音轨。具体命令/接口由 editor 技能提供（如 FFmpeg `-vn` 抽音轨；Palmier 经 MCP 读音轨）。
- **转写**：把音频转为**逐句字幕 + 起止时间戳**（SRT 风格：`句子, start, end`）。本仓库的 `.ai-zh.srt` 就是这种逐句时间轴格式，可直接作为对齐基准。落地转写见 `editor-code-video`（faster-whisper，`vad_filter=True` 顺带去长静音）。
- **产物**：`segments = [{text, start, end}, ...]`。
- **无口播/无字幕的兜底**（混剪、空镜、纯素材）：没有可转写的语音时，改用**静音检测**与**场景检测**定切点——按音频静音段（FFmpeg `silencedetect`/`silenceremove`）找气口与死气，按画面切换（FFmpeg `select='gt(scene,0.3)'`）找镜头边界。产物同样是时间戳列表，后续步骤通用。命令见 `editor-code-video`。
- **精度提醒**：SRT 毫秒用逗号分隔（`HH:MM:SS,mmm`）；秒↔帧换算用四舍五入（round）而非截断，且 fps 必须与落地工程一致，否则音画会系统性漂移。
- **字幕识别的顺序铁律（剪映）**：用剪映"识别字幕"自动生成字幕时，它会识别时间线上**所有**音频片段——务必**先识别字幕、再加 BGM/音效**，否则 BGM 也会被一并识别进字幕。这是软件相关的易错铁律，操作落地见 `editor-other-nle`。

## 三、Step 2：识别断点（什么地方可以/需要切）

逐 segment 分析，标注每个间隙：

| 间隙类型 | 识别依据 | 处理 |
|:---|:---|:---|
| **气口**（呼吸/停顿） | 两句之间的静音间隔；波形"平屏"（满屏无有效语音） | 多数删除，但看衔接（见下气口三规则） |
| **卡顿 / 错口** | 重复词、结巴、明显口误 | 删除 |
| **啰嗦重复** | 语义重复、口癖（"呃""啊"） | 删除 |
| **句界**（正常停顿） | 语义完整处的短停 | 保留，作为镜头切换/贴 B-roll 的天然切入点 |

### 气口处理三规则（课程重点，AI 识别后仍需这把"秤"）
1. **保留**：删掉后衔接不自然的气口要留（例：转折句"但是…"前的气口，删了会突兀）。
2. **扩充**：两个气口挨太近、显得急促时，反而要拉开一点距离，匹配情绪。
3. **叠化垫音**：气口完全去不掉时——分割为前后两段，给音频小把手加一点**叠化**，并在下层垫一小段"安静环境音"，比硬切自然得多。

## 四、Step 3：定切入点 & Step 4：时间轴对齐

- **切入点**：优先选在**句界**和**重音/动作点**，不要切在词中间（会"漏字"出戏）。
- **对齐**：保留下来的 segment 顺次拼接，形成 A-roll 主轨时间轴；记录每段在成片时间线上的 `[t_in, t_out]`。
- **精度**：音频在波形峰值/过零点附近切更干净；落地时由 editor 技能按帧对齐。

## 五、产物（交给后续步骤）

```
audio_timeline = [
  { text:"这一句在讲什么", t_in, t_out, role:"keep",
    gap_before:{type:"气口", action:"保留/删除/叠化"} },
  ...
]
```

- 交给 `03-素材AB分类与替换.md`：按每段 `text` 匹配 B-roll，贴到 `[t_in,t_out]`。
- 交给 `04`：在切点附近用 J/L-cut 让画面与声音错位衔接，更顺。
- 无法干净处理的硬切/空缺 → 标为"黑场待裁"，**交回用户**（不自动删主轨）。

## 六、给"答疑模式"的速答

- **"口播怎么快速剪？"** → 转写成字幕，像改作文一样删掉空白停顿和废话，对应视频段同步删；再按气口三规则微调衔接。
- **"为什么我的口播听着很急/很卡？"** → 多半是气口被一刀切平了。该留的留、太近的扩充、去不掉的用叠化+垫音。
