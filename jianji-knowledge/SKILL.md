# 剪辑知识内核（Knowledge Core）

本内核是这套剪辑技能包的"大脑"，与具体软件无关。它把课程里**人类的操作步骤**重写为**Agent 的决策规则**：
不再说"点哪个按钮"，而是说"在什么情况下、依据什么、做什么决定、输出什么结果"，再由 editor 技能映射到具体软件。

## 内化原则：人类操作 → Agent 技能

| 课程里的人类做法 | 内核里的 Agent 决策 |
|:---|:---|
| 看波形找气口、手动 Ctrl+B 切 | 取带时间戳字幕 → 按规则识别气口/卡顿/重复 → 输出切点列表 |
| 双时间线"筛子法"肉眼捞 B-roll | 用字幕语义关键词检索素材库 → 输出"字幕段 ↔ 候选 B-roll"匹配表 |
| 拖色轮、看示波器调色 | 用示波器/直方图量化判断曝光与白平衡 → 输出参数增量 |
| 凭手感踩点、卡鼓点 | 检测音频节拍/重音时间戳 → 在节拍点对齐切换 |
| 手动拖关键帧、肉眼对齐曲线 | 判断需打帧的参数 → 输出{参数,帧,值,缓动曲线}并保证多参数对齐同步 |
| 肉眼找跟踪特征点、反复重跟 | 判断二维/平面、选占比最大帧、标失败帧处理 → 输出跟踪指令 |

## 七大核心思维（贯穿所有分册）

1. **节奏思维**：剪辑节奏服务于信息传递，信息密度决定镜头时长（信息少→快切，信息多→放长）。
2. **结构思维**：时钟理论可视化节奏（属创作者经验法则），分段式剪辑保证每段独立有趣再总装。
3. **声音思维**：音效/音乐参与叙事，不只是装饰；声音骨架先行。
4. **色彩思维**：非破坏性调色，用示波器量化判断而非纯靠眼；先校正后风格化。
5. **管理思维**：素材命名规范、A/B 分类、demo/export 分离、可移动不离线。
6. **运动思维**：关键帧/变速服务于注意力与情绪——缓入缓出、补帧、变速曲线都为节奏与冲击服务，耗算力操作一律后置。
7. **合成思维**：抠像/蒙版/跟踪/混合模式是"空间重组"——先看元素特征选路（抠/框/叠/跟），再排好"前景/信息/背景"层级。

## 一条总纲

> "掌握工具只是第一步。从剪辑爱好者到专业后期，关键是从**操作工具**转向**构思结构**。"
> 因此本内核优先回答"为什么这么剪"，editor 技能才回答"在这个软件里怎么剪"。
