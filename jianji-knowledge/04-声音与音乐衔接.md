# 04 · 声音与音乐衔接

> 对应课程第1集（自动卡点）与第4集（音频设计）。声音参与叙事，不只是背景。
> 这一册专门覆盖用户常问的"怎么加音乐""怎么让音乐衔接更自然"。

## 一、加 BGM 的标准动作

1. **选乐**：先定情绪与节奏（与视频调性一致）；课程随附 8 首可商用原创音乐可优先用。
2. **铺底**：把 BGM 放到独立音频轨，从头铺。
3. **卡点**：检测音乐**节拍/重音时间戳**（"踩节拍一"=较舒缓/强拍标记，"踩节拍二"=更密集/每拍标记）→ 在节拍点对齐镜头切换/动效。自动卡点工具可一键生成节拍标记。
4. **混音**：BGM 让位人声——口播段压低 BGM（侧链/手动压音量），保证人声清晰；响度统一，避免素材忽大忽小。

## 二、让衔接更自然（高频问题）

| 手法 | 做什么 | 何时用 |
|:---|:---|:---|
| **J-cut**（声先入，轨道呈 J 形） | 下一段**声音先进来**，画面后切（先听后见） | 段落过渡，先用声音预告下一场 |
| **L-cut**（声延续，轨道呈 L 形） | 上一段**声音延续**到下一画面（画面先切、声音后断） | 让对话/旁白盖过剪切点，弱化硬切 |
| **音乐卡在节拍点切** | 镜头切换落在鼓点/重音上 | 混剪、卡点，制造"踩在点上"的爽感 |
| **叠化过渡** | 音频小把手做交叉淡化 | 两段音频/音乐衔接生硬时 |
| **留呼吸** | 高潮间留短暂留白 | 一直满会累，留白让下一波更有力（呼应波峰制） |
| **上升音效(Rise)** | 切换/揭晓前铺渐强音 | 制造期待与紧张 |
| **低频轰鸣(Sub Boom)** | 重点落点加低频冲击 | 强调、转场落地 |

> 一句话答"音乐衔接不自然"：多半是镜头切点没踩在节拍上、且段落之间硬接。**把切点对到节拍/重音，过渡用 J/L-cut 或叠化，段落间留一点呼吸。**

## 三、音效设计（让声音讲故事）

- **环境音/场景音预设**：给画面垫合适环境声，增强临场感；环境声常**提前画面 2-3 秒 + 渐入**（本质是一次 J-cut，先在黑暗里建立场景预期）。
- **J-cut/L-cut** 不止用于音乐，也用于同期声/环境声的进出。
- **音乐自然收尾（复合片段法）**：BGM 长度不匹配时在**副歌结束的重拍**精准切，别只加基础淡出；要做"混响尾迹"自然收束时——截断片段直接加回声无空间扩散，须先复制两段、第二段静音延续，选中两段打包成**音频复合片段**（`Option+G` / `Alt+G`）"骗"软件留出混响扩散空间，再二次修剪只留尾迹精准衔接。
- **自制音效**：录音 → 降调 → 减速 → 加混响（进阶：复制第二层再降调加大厅混响叠加），可造出独特音效。
- **AI 音效**：缺音效时可用 AI 生成补齐（编辑器内置或外部，见落地说明的生成层）。

## 四、口播相关的声音处理（与 02 联动）

- 气口去不掉 → 叠化 + 垫安静环境音（见 `02` 气口三规则）。
- 后期补录：防喷麦、补错字；响度对齐原口播。
- 声道修复：单声道素材修成双声道。

## 五、落地说明

- 检测节拍、抽音轨、做淡化/侧链的**具体命令或接口**由 editor 技能给：
  - Palmier Pro：经 MCP 放置音频片段、调音量/淡化（先查其 MCP 工具）。
  - 代码视频：FFmpeg 做混音/淡入淡出/响度归一（`loudnorm` 两遍法）、BGM 让位人声用**侧链压缩** `sidechaincompress`（人声触发自动压低 BGM）；节拍检测用 `librosa.beat.beat_track` 得 BPM 与节拍时间戳再换算帧号。命令与代码见 `editor-code-video`。
- **生成层（缺素材时补齐）**：旁白用 ElevenLabs TTS；BGM/音效用 fal.ai（音乐、ThinkSound 视频转音效）或 VideoDB `generate_music`/`generate_sound_effect`/`generate_voice`；落地见 `editor-code-video` 与 `fal-ai-media`、`videodb` 技能。生成消耗额度，先与用户确认。
- 本内核只决定**用哪种手法、卡在哪个时间点**。
