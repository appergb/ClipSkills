# 06 · 调色

> 对应课程第9集（一级调色）与第14集（S曲线/色彩克隆）。原则：**非破坏性 + 用示波器量化判断**，不纯靠眼。

## 一、调色工作流（决策顺序）

```
识别素材是否 Log 灰片 → (是)套【技术/转换 LUT】(Log→Rec.709)还原=起点 → 一级校正(曝光/白平衡) → 风格化(色轮/曲线 或叠【创意/风格 LUT】) → 统一多片段(色彩克隆/智能调色)
```

- **Log 灰片**：画面发灰=正常，保留更多动态范围细节。不同厂商不同 Log 曲线（如 Apple Log）。
- **两类 LUT 要分清**：①**技术/转换 LUT**（Log→Rec.709）只做色彩空间还原，**必须最先套**，是调色起点不是终点；②**创意/风格 LUT**（look）决定风格，叠在干净的一级校正之上。导入 LUT 须与素材的 Log 曲线匹配（如 Apple Log 配对应 LUT），不匹配会偏色。
- **非破坏性（铁律）**：调色放在**独立调节轨/调整图层**上，叠在素材上方，随时可改；**不要直接改素材属性**（不可逆）。注意这解决的是"可回退"；而 LUT（尤其 Log→Rec.709 转换）本身的破坏性来自**数据裁切**（高光/阴影/色彩信息丢失），与是否放调整层是两回事——更彻底的非破坏路径是色彩管理（color-managed）工作流。

## 二、用示波器量化判断（Agent 的"眼睛"）

| 示波器 | 看什么 | 判据 |
|:---|:---|:---|
| **波形图** | 亮度分布 | 顶部贴顶=过曝，底部压死=欠曝；把主体亮度落在合理区间 |
| **RGB 分量图** | 三通道亮度 | 三线大致重合 = 白平衡准；某通道偏高=偏色 |
| **矢量图** | 色相 + 饱和度 | 离圆心越远饱和越高；黑/中性点应在圆心（偏离=色偏）；肤色应落在"肤色线"附近* |

> Agent 决策：先用波形/直方图定曝光，再用 RGB 分量/矢量纠白平衡与偏色，最后才谈风格。输出**参数增量**交给 editor 技能落地。

> *肤色线（skin tone line / I-bar）：肤色矢量落在红黄之间、约 **123°（区间 116–126°）**，对所有肤色人种成立（代表皮下血色）。这是通用调色理论补充——课程第9集本身只讲到"圆=饱和、角度=色相、星星飞出方向=偏色方向"，未提肤色线。

## 三、核心工具

- **色彩三要素**：亮度 / 饱和度 / 色相。
- **色轮**（一级核心）：阴影/中间调/高光/偏移四轮；锚点=色相+饱和度，左滑轨=饱和，右滑块=亮度（"左滑轨/右滑块"是剪映专业版色轮的 UI 约定，非行业通用术语）。
- **基础参数**：色温、色调、亮度、对比度、高光、阴影、白色、黑色（默认值均为 0，按需增减）。
- **HSL**：针对单一色相单独调。

## 四、S 曲线（最常用的出片手法）

- 工具：RGB 曲线（曲线工具里最常用的一条），横轴=原亮度（左暗右亮），纵轴=调整后亮度（下暗上亮），**45°斜线=未改变**。
- 做法：**暗部下压 + 亮部上提** → S 形 → 暗的更暗、亮的更亮，增加对比与层次；中间偏左的控制点上/下拖可整体抬/压暗部。
- **RGB 单通道曲线**：分别调红/绿/蓝三通道曲线 = 调色彩平衡（偏色校正的另一条路）。
- 口诀：往上拖变亮，往下拖变暗。

## 五、色彩克隆（统一调性）

- 选未调色片段 → 色彩克隆 → 拖到已调好色的某一帧 → 应用，一键匹配色调。
- 用于 Vlog/多素材让整体色调一致。
- **智能调色工具**（第二选项）：调色新手或快速出片时可用，效果不一定比手动有质感，但能给不错的画面参考、帮助建立调色方向感。

## 六、电影感 & 注意事项

- 电影感常见：略降饱和、暗部偏冷/肤色偏暖、加一点对比的 S 曲线。
- 找喜欢的电影画面当参考，对着调。
- 先曝光白平衡（校正）再风格化；调色是叠在干净一级校正之上的。

## 七、落地说明

- Palmier Pro：经 MCP 加调整层/套 LUT/改色彩参数（先查其工具能力）。
- 代码视频：FFmpeg `curves`/`colorbalance`/`lut3d` 套 LUT 与曲线；Remotion 用 CSS filter/canvas。
- 本内核只产出"该往哪调、调多少"的判断与参数增量。

## 八、给"答疑模式"的速答

- **"调色从哪下手？"** → 先看是不是 Log 灰片套 LUT；再用波形定曝光、RGB 分量正白平衡；最后 S 曲线加对比、色轮定风格；全程放调节轨不破坏原素材。
- **"怎么让一堆素材色调统一？"** → 调好一段当基准，其余用色彩克隆匹配。
