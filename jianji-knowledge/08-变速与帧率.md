# 08 · 变速与帧率

> 对应课程第7集（媒体文件前缀09，动态节奏-曲线变速与素材帧率；注意"文件前缀=集号+2"）。变速不是被动调速，而是**主动用速度控制注意力**——该快快、该慢突然慢，制造冲击。

## 一、变速的本质（先分清两件事）

| 操作 | 改的是什么 | 内容是否丢失 | 用途 |
|:---|:---|:---|:---|
| **变速（Speed Ramp）** | 播放倍率 | **不丢**，只是放快/放慢 | 控制节奏与注意力 |
| 拖片段边缘裁切 | 入出点 | **删内容** | 修剪长度 |

- 进变速工具 → 片段左右出现白色手柄 → 拖拽：**缩短=提高倍速（快放）**，**拉长=降低倍速（慢放）**，默认 1 倍。
- Agent 决策：用户说"这段加速/慢下来"是变速；说"这段太长/砍掉前几秒"是裁切。**别把变速做成裁切**。

## 二、慢放的前置判断：帧率（Frame Rate）够不够（核心红线）

慢放卡顿的根因是**帧率不足**：素材只有 30 帧/秒，慢放把 1 秒内容拉长，原始帧不够填充 → 一顿一顿。

```
有慢放需求 → 查源素材帧率
  ├ 够（如 60/120 帧，慢放后仍≥目标帧）→ 直接慢放，不补帧
  ├ 不够，但必须慢 → 光流法补帧（后期补救，达不到原生）
  └ 运动剧烈/将来还要慢（滑雪等）→ 建议重拍 60/120 帧，别硬补
```

> 根本办法是**前期高帧率拍摄**；补帧只是补救，永远赶不上原生高帧率。

## 三、智能补帧两法对照

| 方法 | 原理 | 优点 | 缺点 | 何时选 |
|:---|:---|:---|:---|:---|
| **帧融合（Frame Blending）** | 前后帧叠加模糊 | 快、省算力 | 运动画面有重影、破碎 | 画面静、赶时间 |
| **光流法（Optical Flow）** | 分析像素运动轨迹算出新中间帧 | 流畅自然 | 耗算力、慢 | 运动画面、要质量（默认优先） |

- Agent 输出补帧决策时写明：`{片段, 是否补帧, 方式=光流/帧融合, 或建议重拍}`。

## 四、曲线变速（自定义曲线）——做"变速转场"

预览窗：**横轴=播放时间，纵轴=变速倍率，三组把手**。向上拖把手=加速，向下=减速。

- **平顶山曲线**（加速 → 匀速 → 减速）= 变速转场，用来无缝填补镜头之间的 1–2 秒空隙。
- **曲线形状预设**（据课程截图，自定义曲线自带，选中即套对应曲线再微调把手）：蒙太奇（尖峰）/ 英雄时刻（双峰）/ 子弹时间（V 型深谷）/ 跳切（单尖峰）/ 闪进（渐升）/ 闪出（渐降）。
- Agent 触发：需要"两镜头之间顺滑过渡 / 节奏推进"→ 输出平顶山曲线，或选对应曲线形状预设。

## 五、动作变速实战——制造冲击（V 谷反差）

1. **定位动作顶点**（如雪墙喷发的最高帧）作为转折中心。
2. **顶点前**曲线拉高 → 快速推进。
3. **顶点瞬间**拉到 **0.3 甚至 0.1 倍** → 超慢，留住细节。
4. 极快到极慢的反差 = 制造动能 + 放大震撼。

> Agent 决策：先在素材里找到动能/情绪顶点的帧时间戳，把它设成倍率谷底；输出 `{片段, 顶点时间戳, 顶点前倍率↑, 顶点倍率≈0.1–0.3}`。

## 六、变速卡点（一键跟鼓点）

导素材 + 加 BGM → 变速里最后一项"**变速卡点**" → 选一个喜欢的卡点效果 → 跟音乐鼓点自动变速。

- 这里的预设是**"动态节奏"分类下的卡点特效模板库**（如 记忆模糊 / 爆闪 / 胶片闪回 / 前后对比 / 堆叠 / 翻页 / 空间翻转 / 拉远 / 震动 / 鱼眼 等），与第四节的"曲线形状预设"是**两套完全不同的东西**，别混（"子弹时间/英雄时刻"那六个属曲线变速，不在变速卡点里）。
- Agent 触发：用户要"跟着音乐节奏变速 / 卡点混剪"→ 选卡点模板、对齐 BGM 重音（重音检测见 `02`/`04`）。

## 七、变速决策总表（Agent 用）

| 需求 | 依据 | 决定 | 输出 |
|:---|:---|:---|:---|
| 慢放 | 源帧率是否足够 | 够→直接慢；不够→光流补帧或建议重拍 | `{片段, 目标倍率, 是否补帧及方式}` |
| 镜头间过渡 | 有 1–2 秒空隙 | 平顶山曲线 | `{片段, 曲线=加速-匀速-减速}` |
| 动作冲击 | 有动能/情绪顶点 | V 谷反差曲线 | `{片段, 顶点时间戳, 倍率关键点}` |
| 跟 BGM | 已加音乐 | 变速卡点选预设 | `{片段, 预设名, BGM}` |

> **适度使用**：变速过多会审美疲劳；优先用在转场空隙与动作高潮处，不要全片乱变。
> 黑场红线：变速涉及的镜头若有硬切/空缺，**不自动删主轨**，标注后交回用户确认（详见 `01`）。

## 八、给"答疑模式"的速答

- **"慢放为什么一卡一卡？"** → 源素材帧率不够；要么用光流补帧救一下，要么重拍 60/120 帧，硬补到不了原生流畅。
- **"怎么做电影感的变速转场？"** → 曲线变速画"平顶山"（加速→匀速→减速），填 1–2 秒镜头空隙。
- **"动作怎么剪得有冲击力？"** → 找动作顶点做转折，顶点前拉快、顶点瞬间拉到 0.1–0.3 倍，极快接极慢制造反差。
- **"想跟着音乐变速？"** → 加好 BGM 后用"变速卡点"选一个卡点特效模板（动态节奏分类），自动跟鼓点；注意"子弹时间/英雄时刻"那六个是**曲线变速**的曲线形状预设，不在变速卡点里。

## 九、落地说明（交给 editor-* 技能）

本册只给"变速曲线形态 / 倍率关键点 / 是否补帧及方式"的判断，按钮级操作下放：

- **Palmier Pro（`editor-palmier-pro`）**：经 MCP 调 `speed`（**先查工具是否存在/参数**再用）；曲线变速、补帧是否支持以官方能力为准，不确定先查 ToolSearch/官方文档。
- **代码视频（`editor-code-video`，FFmpeg·Remotion）**：匀速变速 `setpts=0.5*PTS`（视频）+ `atempo`（音频）；**非线性曲线 FFmpeg 难做**，需分段拼接或用 Remotion 逐帧控制倍率；补帧 `minterpolate=mi_mode=mci`（慢、近似），专业补帧建议外接达芬奇/Topaz。
- **剪映·其他 NLE（`editor-other-nle`）**：剪映用变速面板 + 智能补帧（帧融合/光流）+ 变速卡点预设；其他 NLE 的对应能力先查再用，不臆造软件功能。
